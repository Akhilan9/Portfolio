# 🌐 Akhilan Portfolio Website

## 🚀 Modern Developer Portfolio with Advanced Animations

This is a fully responsive and modern personal portfolio website developed to showcase:

* 💻 Technical Skills
* 🛡️ Cybersecurity Projects
* 🤖 AI & ML Projects
* 📜 Certifications
* 📂 Resume
* 📞 Contact Information
* 🌟 Achievements & Social Presence

The portfolio is designed with smooth animations, modern UI/UX principles, glassmorphism effects, and interactive sections to create a professional online presence.

# 🚀 Installation Guide

## 1️⃣ Clone Repository

```bash
git clone <your-github-repository-link>
cd akhilan-portfolio
```

---

# 📦 Install Dependencies

```bash
npm install
```

---

# ▶️ Run Development Server

```bash
npm run dev
```

The application will run at:

```bash
http://localhost:5173
```

---

# ⚙️ Build for Production

```bash
npm run build
```
